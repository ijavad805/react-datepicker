# React Datepicker

A customizable datepicker and calendar component library with English and Persian support for React applications.

## Documentation

Full documentation for the Datepicker and Calendar components is available on the Docusaurus site:

- [Datepicker Guide](https://ijavad805.github.io/react-datepicker/docs/datepicker)
- [Calendar Guide](https://ijavad805.github.io/react-datepicker/docs/calendar)

For local development, navigate to the `documents` directory and run the Docusaurus commands listed there.
